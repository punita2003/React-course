## How It Works
-User types a message in the input field
-Message is stored using React `useState`
-Bot generates a predefined response-:
  a) What Date is today?
  b) About Flip a coin?
  c) About roll a Dice?
  d) Thankyou message..
- Chat updates dynamically without page refresh
